App Installation

After, Purchase app you need to install our app.

Create and Configure Brand to Products

1. For creating/viewing all brands, Goto Sales -> Configurations -> Products -> Product Brand menu. Here you can see all created brands and you can create new brands.
2. For configure Brand to Products, Goto Products screen and there is option Product Brand where you can set Brand to each products.
